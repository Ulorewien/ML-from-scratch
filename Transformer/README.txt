Instructions to Run the Code
===========================================================

To run the code, use the following command in your terminal:

    python main.py

You will be prompted to choose a task from the following options:

    Choose a task:
    1. Classification Task
    2. Language Modelling
    3. Exploration
    4. Exit
    Enter a number:

Enter the corresponding number to run the desired task:

1. **Classification Task**: Runs the part of the code focused on classification.
2. **Language Modelling**: Runs the part of the code related to language modeling.
3. **Exploration**: Runs the exploratory part of the assignment.
4. **Exit**: Exits the program.


Additional Notes
===========================================================

- Make sure to have your datasets in the appropriate directory if required.
